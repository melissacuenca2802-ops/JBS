Bem-vindo ao Transportal!

Este portal foi desenvolvido para facilitar o acesso às informações de transporte para todas as pessoas. Aqui você pode consultar, de forma clara e rápida, os trajetos, horários e pontos de parada. O portal é pensado para manter os dados de horários e trajetos sempre atualizados, garantindo mais segurança e organização no planejamento do seu deslocamento. Este formato de consulta foi criado para ser simples, visual e acessível, oferecendo autonomia para que todos possam acompanhar os horários sem dificuldade.

Como usar o portal

  1. Selecione o bairro desejado
  2. Visualize os horários
  3. Confira os pontos do trajeto
  4. Informações sempre atualizadas
